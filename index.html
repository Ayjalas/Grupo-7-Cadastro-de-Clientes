Bem-vindo ao Sistema de Cadastro de Clientes

Este sistema permite gerenciar seus clientes com as operações:

  • Cadastrar – adicionar novos clientes
  • Listar – visualizar, editar, excluir e buscar clientes
  • Login – proteção das funcionalidades

Utilize o menu lateral para navegar.

Os dados são salvos permanentemente no navegador (LocalStorage).
